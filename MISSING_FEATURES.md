This document lists features that are currently missing from Aura Studio and could be added in the future.

## üéØ High Priority Missing Features

### 1. **Due Dates & Deadlines**

- Add due dates to songs and projects
- Add deadlines to individual tasks
- Calendar view showing upcoming deadlines
- Overdue indicators and notifications
- Timeline view with deadlines

### 2. **Priority Levels**

- Priority system for songs (Low, Medium, High, Urgent)
- Priority system for tasks
- Filter and sort by priority
- Visual priority indicators (colors, badges)

### 3. **Task Dependencies**

- Mark tasks as dependent on other tasks
- Visual dependency chains
- Block completion of dependent tasks until prerequisites are done
- Automatic task ordering based on dependencies

### 4. **Time Tracking**

- Track time spent on songs/projects
- Time estimates vs actual time
- Time logging per task
- Time reports and analytics

### 5. **File Attachments**

- Upload and attach files to songs
- Attach files to tasks
- Version file attachments
- File preview and download
- Integration with cloud storage (Drive, Dropbox)

### 6. **Search Functionality**

- Global search across all songs, projects, tasks, notes
- Advanced search filters
- Search by tags, status, date ranges
- Quick search bar in header

### 7. **Activity Log / History**

- Track changes to songs/projects
- See who made changes (if multi-user)
- Revert changes
- Activity timeline per song/project

### 8. **Templates**

- Song templates with pre-filled tasks
- Project templates
- Custom templates
- Quick start from template
